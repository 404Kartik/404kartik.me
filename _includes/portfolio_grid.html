Project Description

Hit the ball is a small javascript project used to learn new JS libraries for e.g : Paper JS. You can use your ARROW KEYS to control the main ball and use SPACEBAR to shoot. The faster you shoot at the counter, you will get more points as the counter descends and when it approaches zero it starts to go at a new position. Check the live demo deployed at Github pages.
